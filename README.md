# Herb's House Partner Portal

A resource library portal for Herb's House Coffee wholesale partners.

## Quick Start

```bash
npm install
npm run dev
```

## Documentation

See [project-spec.md](./project-spec.md) for complete design and implementation specifications.

## Overview

- **Purpose**: Centralized resource hub for wholesale partners
- **Auth**: Shared company passcode (24-hour sessions)
- **Sections**: Marketing, Sales, Projects
- **Stack**: React, localStorage, GitHub Pages
